CS AI Assistant
with Voice

음성 인식(STT)과 RAG 기술을 결합한 24/7 AI 상담원.
복잡한 매뉴얼도 즉시 학습하여 정확도 높은 답변을 제공하고, 예약 접수까지 스스로 처리합니다.

Key Features

Real-time Voice Conversation

텍스트뿐만 아니라 고객의 음성을 실시간으로 인식하여 자연스러운 대화가 가능합니다. 전화 응대 업무를 완벽하게 자동화하세요.

Manual Learning (RAG)

기존의 방대한 매뉴얼(PDF, Word)과 FAQ 문서를 AI가 학습합니다. 할루시네이션 없이 기업의 규정에 맞는 정확한 답변만 제공합니다.

Auto Reservation System

상담 중 예약이나 일정이 필요한 경우, 캘린더와 연동하여 AI가 즉시 빈 시간을 확인하고 예약을 확정짓습니다.

How It Works

1. 고객 문의 접수 (전화/채팅)

고객이 전화나 채팅으로 문의하면 AI가 즉시 응대 모드로 진입하여 음성을 텍스트로 변환(STT)하고 의도를 파악합니다.

2. 지식 검색 및 답변 생성

내부 지식 베이스(매뉴얼)에서 최적의 답변을 검색(Retrieval)하고, 자연스러운 문장으로 생성하여 음성(TTS)으로 답변합니다.

3. 액션 수행 (예약/이관)

예약이 필요한 경우 시스템에 직접 등록하며, AI가 해결하기 어려운 감정적/복합적 문제는 상담원에게 매끄럽게 연결합니다.
